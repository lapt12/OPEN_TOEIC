# 📖 TOEIC単語テストBot

![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python&logoColor=white)
![discord.py](https://img.shields.io/badge/discord.py-2.x-5865F2?logo=discord&logoColor=white)
![License](https://img.shields.io/badge/license-Private-lightgrey)

Discordのユーザーインストール型アプリとして動作する、TOEIC単語学習Botです。  
1250語をQuiz1〜Quiz10の10セットに分け、4択クイズ形式で学習できます。進捗・正答率・苦手単語が自動で記録されるので、繰り返し使うほど自分の弱点が見えてきます。

---

## ✨ 主な機能

| 機能 | 説明 |
|------|------|
| **4択単語クイズ** | 英→日 / 日→英 の両モードに対応 |
| **進捗トラッキング** | 各クイズセットの正答率・学習済み単語数を記録 |
| **週数カウント** | 1セット（125語）を一周するごとにカウントアップ |
| **テスト範囲切り替え** | Quiz単位 or 全1250語から自由に選択可能 |
| **パスワード認証** | 招待制のため、配布されたパスワードで初回認証 |

---

## 🚀 使い方

### 1. Botをインストール
サーバーへの追加ではなく**ユーザーインストール**型です。Botを追加することで、どのサーバーや DM からでも利用できます。

### 2. コマンドを実行

```
/test
```

はじめて使う場合は認証画面が表示されます。配布されたパスワードを入力してください。

### 3. テストを開始

認証後、ホーム画面が表示されます。

```
[ テスト範囲を選択 ] プルダウン
  └─ Quiz1〜Quiz10 / all（全範囲）から選択

[ ✈ Start ] ボタン
  └─ クイズ開始！
```

### 4. クイズに答える

- 問題文の単語に対応する選択肢（1〜4）を押す
- 正解・不正解がその場で表示される
- **「次へ」** で次の問題へ / **「タイトルへ」** でホームに戻る

### 5. モード切り替え

クイズ画面上部の ⚙ ボタンで **英→日 / 日→英** をいつでも切り替えられます。

## 📊 ホーム画面の見かた

ホーム画面には現在の学習状況がひと目でわかるサマリーが表示されます。

```
Status
├─ 全ての合計週数       : 今まで何周したか
├─ 進捗                : 学習済み語数 / 総語数（%）
├─ 総合回答数          : これまでの回答総数
├─ 正答率              : 正解した割合（%）
└─ 苦手登録単語数      : 苦手リストに入っている単語数
```

進捗はサムネイル画像でも視覚的に確認できます。

## 📝 注意事項

- 本Botは**招待制**です。パスワードをお持ちでない方はご利用いただけません。
- Botがオフラインの場合、コマンドが応答しないことがあります。
